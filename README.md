# lang-flow_3-d-digital-human_-back

这是一个基于 Flask 的后端项目，主要用于构建数字人类应用。该项目结合了文档处理、向量存储、模型调用等功能，适用于需要进行大规模文本处理和交互的应用场景。

## 功能特性

- **文档处理**：支持文档加载、分块、向量化存储。
- **对象存储服务集成**：集成了 OSS 文件上传功能，支持同步与异步文件上传。
- **模型调用**：支持调用外部模型接口进行对话补全。
- **模块化设计**：通过模块化路由和功能组件，便于扩展和维护。

## 目录结构

- `src/`：核心源代码目录。
  - `file/`：文件处理相关模块。
  - `pojo/`：数据模型定义。
  - `route/`：路由注册模块。
  - `utils/`：工具类模块，包括文档分块、向量化、模型调用等。
- `database/`：数据库配置文件。
- `vectorstores/`：向量存储目录，包含多个索引文件。

## 安装与运行

### 依赖安装

确保已安装 Python 3.x 和 pip，然后执行：

```bash
pip install -r requirements.txt
```

### 启动服务

运行 `main.py` 启动 Flask 服务：

```bash
python main.py
```

## 使用说明

1. **文档处理**：通过 `/file` 接口上传文档，系统会自动进行分块和向量化处理。
2. **模型调用**：通过 `/agent` 接口调用模型进行对话补全。
3. **知识库构建**：通过 `/kbs` 接口构建知识库并存储至向量数据库。
